Project Description:
Python Jump Quest is a side-scrolling platformer game written on python using cmu graphics library. The player controls a character navigating through an automatically generated landscape. Goal of the Player would be navigating through a screen, collecting coins, overcoming obstacles & enemies. It can also acquire power-ups for additional abilities. 


How to Run the Project:
1. Install the cmu_graphics libraries.
2. Ensure that all the needed image files (like coins.png, Ground.png) are accessible in the same folder as the main code file.
3. Run the main file of your project
4. Use the keyboard to control the player character.




How to install any needed libraries:
No libraries except for cmu_graphics.


A list of any shortcut commands that exist:
You can press R or r to restart the game.
